

ATM Interface – Java Console Application:

 Description:

The ATM Interface is a simple Java console-based project that simulates basic ATM operations like:

Checking account balance

Depositing money

Withdrawing money

PIN authentication



It uses Object-Oriented Programming (OOP) concepts such as classes, methods, and encapsulation.


 Features:

1. PIN-based user authentication

2. Check current account balance

3. Deposit money

4. Withdraw money (with balance validation)

5. Simple menu-driven interface

6. Initial balance set by default



 Technologies Used:

1.Java (JDK 8 or later)

2.Scanner class for input

3.Basic console-based UI



 How to Run ?

1. Clone or download the project folder.


2. Open the project in any Java IDE (Eclipse, IntelliJ, VS Code) or a simple text editor.


3. Compile and run the ATMInterface.java file.



 Concepts Demonstrated:

Classes and Objects

Access Modifiers (public, private)

Conditional Statements (if-else, switch-case)

Loops (do-while)

Input/Output using Scanner

Method creation and reuse
